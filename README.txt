# GG Political AI Lab

Ez a csomag egy többoldalas mini site GitHub Pages-hez.

## Fájlok
- index.html -> menő főoldal
- ner-generator.html -> ultimate NER generátor
- propaganda3000.html -> final boss Propaganda 3000

## GitHub Pages
Ha ezeket a fájlokat ugyanabban a public repo-ban a gyökérbe töltöd fel, akkor:

- / -> főoldal
- /ner-generator.html -> NER generátor
- /propaganda3000.html -> Propaganda 3000

## Javaslat
A jelenlegi repóban egyszerűen cseréld le az index.html-t erre a főoldalra, és töltsd fel mellé a másik két HTML-t.
